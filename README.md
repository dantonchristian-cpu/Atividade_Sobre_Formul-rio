Calculadora de Média Escolar — LPWI

Este projeto é uma aplicação web completa desenvolvida como atividade prática individual para a disciplina de Linguagem de Programação para Web I (LPWI).

📋 Funcionalidades Implementadas

Validação de Formulários: Restrição de notas entre 0 e 10 com suporte a decimais diretamente na interface.

Cálculo de Média no Servidor: Processamento dinâmico das notas enviadas para o backend em Node.js com Express, calculando a média e retornando a classificação exata (Aprovado, Exame Final ou Reprovado).

Histórico Dinâmico: Alimentação dinâmica da tabela de resultados na interface através dos retornos JSON do servidor.

Persistência de Dados: Uso da API localStorage para salvar os dados dos alunos localmente no navegador, impedindo a perda de registros ao atualizar a página.

Limpeza de Registros: Opção para limpar de forma dinâmica e instantânea o histórico persistido.
